Timetable

List That should be followed in order

  1. Wake up at 6:30
  2. Offer Prayer

  3. Eat Breakfast
  4. Get ready
  5. Start your day

    6. Good luck!

List That can be followed without any order

  • Wash clothes
  • Eat Fruit

Description of above lists
