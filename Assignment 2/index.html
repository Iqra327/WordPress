Timetable

There are two timetables, One should be followed step by step and the other can be followed without any order.

List That should be followed in order

  1. Wake up at 6:30
  2. Offer Prayer
    1. Recite Quran
  3. Eat Breakfast
  4. Get ready
  5. Start your day

    6. Good luck!

List That can be followed without any order

  • Wash clothes
  • Eat Fruit

Description of above lists

Ordered list
This list is in ordered form because as a muslim you should follow some of these steps and the other steps are good for yourself.
Un Ordered List
This list is not in order because these things can be done without following any order
